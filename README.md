# Executive Sales Performance Dashboard

## Project Overview

This project demonstrates how retail sales data can be transformed into an executive business intelligence dashboard using Microsoft Excel, Power Query and Power BI.

The dashboard will help business owners and managers monitor sales performance, profitability, products, customers, payment methods and business trends.

## Business Problem

Retail businesses often collect sales information but struggle to convert it into clear and timely management insights. Manual reporting can be slow, inconsistent and difficult to interpret.

This project will create an interactive reporting solution that helps decision-makers understand business performance quickly.

## Project Objectives

- Analyse sales and profit performance
- Identify top-performing products and categories
- Monitor monthly and regional sales trends
- Understand customer purchasing behaviour
- Compare payment methods
- Create clear executive KPIs
- Reduce manual reporting work
